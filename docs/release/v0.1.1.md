# Release v0.1.1

## Summary
- Added global `--help`/`--version` and subcommand help.
- Added `describe` topics: `ai-authoring`, `params`.
- Added `ls` command for listing IDs and vocab.

## Added
- `ideamark --help` and `ideamark <cmd> --help`
- `ideamark --version`
- `ideamark describe ai-authoring`
- `ideamark describe params [--format json|yaml|md]`
- `ideamark ls [<infile>|-] [--sections] [--occurrences] [--entities] [--vocab] [--format json|md]`

## Notes
- v0.1.0 behaviors are preserved; this is a delta release.
