# IdeaMark CLI v0.3.1

v0.3.1 is an additive patch release for IdeaMark Core v1.2.0 draft support.

## Highlights

- Adds known optional `skeletons` namespace support with Core-mode warning diagnostics.
- Adds `ideamark ls --skeletons` summaries for Skeleton Graph discovery.
- Extends `describe` capabilities, checklist, and vocab output for Skeleton Graph and Projection Profile discovery boundaries.
- Adds a small retrieval fixture harness that compares required slots/links against document-side skeletons. This is not a retrieval engine, ranking system, or reconstruction-quality evaluator.
- Updates describe guide JSON and package metadata to tool version `0.3.1` and contract `1.2.0-draft.2`.
