# IdeaMark Params

This file summarizes normalization constraints for tooling.

- `section_id` format: `SEC-*`
- `occurrence_id` format: `OCC-*`
- `entity_id` format: `IE-*`
- `anchorage.view` and `anchorage.phase` should use the published vocab.
- `status.state` should be one of the published values.
