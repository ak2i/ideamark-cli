# Part 5 — Projection Specification

**Version:** IdeaMark Core v1.2.0  
**Status:** Planned

This part will define Projection as an external but interoperable specification.

Projection is expected to include concepts such as:

- search tasks
- entity model
- coordinate preferences
- authoring context
- evaluation criteria

Projection should be treated as a reusable intellectual asset, not merely a prompt parameter.
